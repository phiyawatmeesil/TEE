เเนะนำสถานที่ท่องเที่ยว

ที่เที่ยวยอดนิยม 4 อันดับจุดหมายปลายทางของโลก

  • Get Started

1. อิสตันบูล, ตุรกี

อิสตันบูล ที่เที่ยวยอดนิยมอันดับ 1 เดิมชื่อ คอนสแตนติโนเปิล เป็นเมืองที่มีประชากรมากที่สุดในประเทศตุรกี มีความสำคัญพอๆ กับเมืองหลวงของตุรกี โดยเคยเป็นเมืองหลวงของตุรกีมายาวนานถึง 1,610 ปี ก่อนที่จะย้ายไปยังเมืองอังการา

  • Learn More

2. โรม, อิตาลี

กรุงโรม อาณาจักรโรมันอันยิ่งใหญ่สุดคลาสสิค ที่มีอารยธรรมที่รุ่งเรืองมาตั้งแต่อดีตกว่า 2 พันปี จะเที่ยวกรุงโรมให้ทั่วภายในวันเดียวคงไม่พอ เป็นที่เที่ยวยอดนิยมที่มีสถานที่สำคัญๆ ให้เที่ยวชมมากมาย ไปโยนเหรียญเสี่ยงทายที่น้ำพุเทรวี่

  • Learn More

3. ลอนดอน, สหราชอาณาจักร

ลอนดอน เมืองที่มีศูนย์กลางทางธุรกิจที่สำคัญแห่งหนึ่งของโลก เปรียบเสมือนแหล่งรวมประวัติศาสตร์ศิลปะ แฟชั่น อาหาร และเบียร์แบบอังกฤษ ท่ามกลางสถาปัตยกรรมอันยิ่งใหญ่

  • Learn More

    • 4. ปารีส, ฝรั่งเศส

    ปารีส เป็นที่เที่ยวยอดนิยม ที่คับคั่งไปด้วยนักท่องเที่ยว ที่ต่างพากันหลั่งไหลมาชมหอไอเฟลที่ตั้งตระหง่าน ศิลปะที่งดงามที่พิพิธภัณฑ์ลูฟวร์ และสถาปัตยกรรมอันเก่าแก่ที่นอร์ทเทอดาม นั่งเพลินๆ
